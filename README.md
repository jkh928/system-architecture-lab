# System Architecture Lab

## 🚀 Overview
This repository serves as the central implementation hub for my studies in the **WGU BS-MS in Computer Science (AI Track)**. It bridges the gap between theoretical coursework and functional software engineering.

## 🛠️ Core Protocols
- **Complexity Filter:** Every algorithm is documented with its Big O time and space complexity.
- **Data Shape Verification:** Strict validation of matrix dimensions and object structures before execution.
- **Error-Resistant Design:** Utilization of `try/except` blocks and modular helper functions for robust flow control.

## 📂 Structure
- `/foundations`: Core Python logic, error handling, and functional programming.
- `/data_structures`: Implementations of Linked Lists, Graphs, and Trees.
- `/applied_algos`: Real-world applications of Pathfinding (Dijkstra's) and Graph Theory.
